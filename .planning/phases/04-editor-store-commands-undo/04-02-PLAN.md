    autosave.ts exports:
    - `Autosave` interface: `{ flush(): Promise<WriteResult>; dispose(): void }`
    - `createAutosave(store: Store, path: string, delayMs?: number): Autosave`
      - default delayMs = 500
      - Internal: `let timer: NodeJS.Timeout | null = null`
      - `doWrite = async () => writeSpecFile(path, store.getState().spec, store.getState().astHandle)`
      - `schedule = () => { if (timer) clearTimeout(timer); timer = setTimeout(() => { timer = null; void doWrite(); }, delayMs) }`
      - `flush = async (): Promise<WriteResult> => { if (timer) { clearTimeout(timer); timer = null; } return doWrite(); }`
      - `const unsubscribe = store.subscribe(() => schedule())`
      - `const beforeExitHandler = () => { void flush(); }`
      - `process.on("beforeExit", beforeExitHandler)` registered on creation
      - `dispose() { if (timer) clearTimeout(timer); unsubscribe(); process.off("beforeExit", beforeExitHandler); }`

    tests/autosave-debounce.test.ts covers (vi.useFakeTimers() novel pattern):
    - "trailing-edge only": 1 apply → vi.advanceTimersByTime(100) → writeSpy called 0 times → vi.advanceTimersByTime(500) → writeSpy called 1 time
    - "coalesce": 10 applies in 100ms (each 10ms apart via vi.advanceTimersByTime) → vi.advanceTimersByTime(500) → writeSpy called exactly 1 time (SERDE-06 + EDITOR-04)
    - "flush": apply, then flush() before timer fires → writeSpy called 1 time immediately
    - "beforeExit handler": spy on process.on, verify "beforeExit" is registered; spy on process.off in dispose()
    - "dispose clears timer": apply, dispose() before timer fires → vi.advanceTimersByTime(1000) → writeSpy called 0 times
