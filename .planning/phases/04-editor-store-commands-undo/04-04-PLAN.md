    - argsSchema: z.object({ screenId: ScreenIdSchema, variantKind: z.enum(["content","empty","loading","error"]), fromPath: JsonPointerSchema, toParentPath: JsonPointerSchema, toIndex: z.coerce.number().int().min(0) })
    - apply: capture node JSON; delete from fromPath; insert at toParentPath[toIndex]; AST mirrors spec ops; inverseArgs = { screenIndex, variantKind, fromPath, toParentPath, toIndex, movedNodeJSON }
    - invert: reverse the move

    reorder-component:
    - argsSchema: z.object({ screenId: ScreenIdSchema, variantKind: z.enum(["content","empty","loading","error"]), parentPath: JsonPointerSchema, fromIndex: z.coerce.number().int().min(0), toIndex: z.coerce.number().int().min(0) })
    - apply: same-parent reorder; splice from fromIndex insert at toIndex; inverseArgs = { fromIndex: toIndex, toIndex: fromIndex } (reverse indices)
    - invert: reorder back with swapped indices
